# Wordrill 디자인 시스템

이 문서는 Wordrill 웹 UI에서 반복해서 사용하는 시각 언어와 상호작용 상태의 의미를 정의한다. 구현 세부 값은 `app/globals.css`의 CSS 변수와 컴포넌트를 기준으로 하며, 공통 규칙이 추가되거나 변경될 때 이 문서를 함께 갱신한다.

## 기본 원칙

- 같은 의미에는 같은 시각 표현을 사용한다.
- 현재 위치, Hover, Focus, 기능 활성화, 알림 상태를 서로 다른 표현으로 구분한다.
- 색상만으로 상태를 전달하지 않고 형태, 위치, 텍스트 또는 접근성 속성을 함께 사용한다.
- Lucide 아이콘은 기본적으로 선형 형태를 유지한다. 내부 채움은 현재 위치가 아니라 토글 기능의 활성 상태처럼 명확히 다른 의미에만 사용한다.

## 좌측 네비게이션

### 현재 위치

현재 페이지 링크에는 다음 표현을 함께 사용한다.

- `3px` accent 색상의 좌측 세로 라인
- `accent-soft` 배경
- accent 색상의 아이콘과 텍스트
- 굵은 메뉴 이름
- `aria-current="page"`

메뉴가 접혀 있을 때는 좌측 라인, 배경, 아이콘 색상으로 현재 위치를 구분한다. 메뉴가 펼쳐지면 같은 표현에 텍스트 강조가 추가된다. `/rooms/[roomId]`처럼 하위 경로에 있어도 상위 메뉴인 `/rooms`를 현재 위치로 표시한다.

### 상호작용 상태

| 상태 | 표현 | 의미 |
| --- | --- | --- |
| 기본 | 투명 배경, 기본 아이콘·텍스트 색상 | 이동할 수 있는 메뉴 |
| Hover | 중립적인 `hover` 배경과 약한 테두리 | 포인터가 가리키는 항목 |
| Focus | accent 색상의 focus ring | 키보드로 선택한 항목 |
| 현재 위치 | 좌측 라인, `accent-soft` 배경, accent 아이콘·텍스트 | 현재 보고 있는 페이지 |
| 토글 활성 | 내부가 채워진 아이콘과 accent 색상 | 테스트 모드처럼 켜고 끄는 기능의 활성 상태 |
| 알림 | 메뉴 오른쪽의 배지 또는 점 | 읽지 않은 항목이나 주의가 필요한 상태 |

현재 위치에 점을 사용하지 않는다. 점은 온라인 상태나 읽지 않은 알림으로 오해할 수 있다. 현재 위치 아이콘도 채우지 않는다. Lucide 아이콘마다 채움 결과가 달라질 수 있고 토글 활성 상태와 의미가 겹치기 때문이다.

## 색상 토큰

주요 색상은 `app/globals.css`의 CSS 변수로 관리한다.

- `--accent`: 선택, 현재 위치, 주요 동작
- `--accent-soft`: 선택된 항목의 약한 배경
- `--hover`: 포인터 Hover 배경
- `--focus`: 입력 요소의 focus 강조
- `--line`: 경계와 구분선
- `--muted`: 보조 정보

라이트·다크 모드에서 같은 의미의 토큰을 사용하고 실제 색상 값만 테마별로 변경한다.
